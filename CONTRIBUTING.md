# Contributing to Agronav

Thank you for your interest in contributing to Agronav! We welcome contributions from the community to help improve this autonomous navigation framework.

## How to Contribute

1.  **Fork the Repository**: Click the "Fork" button at the top right of this page.
2.  **Clone your Fork**: `git clone https://github.com/YOUR_USERNAME/agronav.git`
3.  **Create a Branch**: `git checkout -b my-new-feature`
4.  **Make Changes**: Implement your feature or fix.
5.  **Commit Changes**: `git commit -m "Add some feature"`
6.  **Push to Branch**: `git push origin my-new-feature`
7.  **Submit a Pull Request**: Go to the original repository and click "New Pull Request".

## Guidelines

*   Follow the existing code style.
*   Update documentation if necessary.
*   Ensure your code runs without errors.
*   Be respectful and constructive in discussions.

## Reporting Issues

If you find a bug or have a suggestion, please open an issue in the [Issues](https://github.com/bahattinyunus/agronav/issues) tab.
